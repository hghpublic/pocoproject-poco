=== building

See README.md.

To build poco with cmake::
+
[source, shell]
----
$ mkdir cmake-build
$ cd cmake-build
$ cmake ..
$ cmake --build . --config Release
----

switches::
+
[source, cmake]
----
ENABLE_TESTS
ENABLE_SAMPLES
----
